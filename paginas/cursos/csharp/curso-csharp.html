  Os desenvolvedores da microsoft programavam em C++ e a linguagem do mercado era VB(Visual Basic), os mesmos não queriam utilizar o VB, embora ele tivesse mais recursos que o C++ naquela época, então, um grupo de desenvolvedores resolveu criar o C#(CSHARP), alguns dizem que o # é pelo C++ ter evoluído para o C++++ formando o jogo da velha. Quando o c# foi criado, tinha como objetivo ser a melhor linguagem de programação, desta forma, a justificativa etimologica de c#, foi ligada a música pelo seguinte motivo: a primeira nota musical é o Dó que é representado pela letra C, todas as notas musicais estão no mesmo nível, sendo assim, quando queremos que uma nota fique acima da outra dizemos que a nota é sustenida, no caso, dó sustenido ficaria C#.
